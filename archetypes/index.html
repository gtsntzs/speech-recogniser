Maven Archetypes

A scalable, modular speech recogniser.

Creating new sphinx4soc modules using Maven Archetypes. The archetypes currently offered are:

  • archetype-model - generates a module where the models, dictionary files etc. are packed in a jar for easy deployment.
  • archetype-component - generating new sphinx4soc camel components.
  • and archetype-app - to create new sphinx4soc apps similar to the speech-recognition-app project.

A tutorial on how to create new archetypes can be found here.
