Project Modules

This project has declared the following modules:

Name Description
CMU Sphinx4soc Core Library The CMU sphinx Library.The official library API has some restrictions in method visibility, therefore cannot be extented to fit the needs of a service oriented approach.
Sphinx4soc Library Sphinx4soc Library contains mainly helper classes for better integration with Apache Camel.
Camel Delta Features Component Sphinx4soc Camel Delta Features Component calculates the delta of the mfcc features.
Camel Windower Component Sphinx4soc Camel Windower Component applies the window function over the audio signal.
Camel Microphone Component Sphinx4soc Camel Microphone Component captures the input of the microphone.
Sphinx4 Camel Components Sphinx4soc Camel Components.
Sphinx Models A simple jar container of the models loaded by the recogniser.
Sphinx4soc :: Parent Sphinx4soc Parent Project.
